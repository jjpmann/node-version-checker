# Node Version Checker

Check node version against required versions.


## Install

```bash
npm install --save-dev node-version-checker
```

## Example

```javascript
var nvc = require('../../git/nvc/index.js');  
nvc(5);
```

Output

```
------------------------------------------------------

Node version does not match, version 5 required.

------------------------------------------------------
````
